RSS Feed Checker and Telegram Notifier

This script periodically checks a RSS feed for new entries and sends updates to a specified Telegram chat. It is designed to run as a cron job and uses asynchronous operations to efficiently handle network requests. The application runs in a docker container on a hetzner cloud server and will be updated and redeployed via a CI/CD pipeline in GitHub
